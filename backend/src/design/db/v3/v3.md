# v3 — Reviews, bookmarks, notes, comments, notifications

## Changes

### `users` table (altered)
- **Added**: `bio TEXT` — user biography
- **Added**: `avatar_url TEXT` — profile picture URL

### `reviews` table (new)
| Column | Type | Notes |
|---|---|---|
| id | UUID PK | gen_random_uuid() |
| user_id | UUID FK | → users.id, CASCADE |
| course_id | UUID FK | → courses.id, CASCADE |
| rating | INTEGER | CHECK 1-5 |
| comment | TEXT | nullable |
| created_at | TIMESTAMPTZ | |
| updated_at | TIMESTAMPTZ | |
| | UNIQUE | (user_id, course_id) |

### `bookmarks` table (new)
| Column | Type | Notes |
|---|---|---|
| user_id | UUID FK | → users.id, CASCADE |
| course_id | UUID FK | → courses.id, CASCADE |
| created_at | TIMESTAMPTZ | |
| | PK | (user_id, course_id) |

### `notes` table (new)
| Column | Type | Notes |
|---|---|---|
| id | UUID PK | gen_random_uuid() |
| user_id | UUID FK | → users.id, CASCADE |
| lesson_id | UUID FK | → lessons.id, CASCADE |
| content | TEXT | NOT NULL |
| timestamp_sec | INTEGER | Video position in seconds, default 0 |
| created_at | TIMESTAMPTZ | |
| updated_at | TIMESTAMPTZ | |

### `comments` table (new)
| Column | Type | Notes |
|---|---|---|
| id | UUID PK | gen_random_uuid() |
| user_id | UUID FK | → users.id, CASCADE |
| lesson_id | UUID FK | → lessons.id, CASCADE |
| parent_id | UUID FK | → comments.id, CASCADE, nullable (for threaded replies) |
| content | TEXT | NOT NULL, may contain @[username] mentions |
| created_at | TIMESTAMPTZ | |
| updated_at | TIMESTAMPTZ | |

### `comment_mentions` table (new)
| Column | Type | Notes |
|---|---|---|
| comment_id | UUID FK | → comments.id, CASCADE |
| mentioned_user_id | UUID FK | → users.id, CASCADE |
| | PK | (comment_id, mentioned_user_id) |

### `notifications` table (new)
| Column | Type | Notes |
|---|---|---|
| id | UUID PK | gen_random_uuid() |
| user_id | UUID FK | → users.id, recipient, CASCADE |
| type | VARCHAR(50) | 'mention', 'reply', 'review' |
| reference_id | UUID | comment_id or review_id |
| reference_type | VARCHAR(50) | 'comment', 'review' |
| message | TEXT | Display text |
| is_read | BOOLEAN | default false |
| created_at | TIMESTAMPTZ | |

## Reason

Adding social and engagement features to transform the platform from a basic course viewer into a collaborative learning environment. Reviews provide social proof, bookmarks enable course discovery, notes support personal learning, and comments with @mentions enable peer-to-peer discussion.
