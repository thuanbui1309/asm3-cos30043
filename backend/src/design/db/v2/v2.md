# v2 — lessons: replace duration with description

## Changes

### `lessons` table
- **Removed**: `duration INTEGER DEFAULT 0`
- **Added**: `description TEXT`

## Reason

Course creation form was updated to collect a text description per lesson instead of a numeric duration in minutes. Duration is not displayed anywhere in the UI and is not required for the learning experience.

## Tables after v2

### `lessons`
| Column | Type | Notes |
|---|---|---|
| id | UUID PK | gen_random_uuid() |
| course_id | UUID FK | → courses.id, CASCADE |
| title | VARCHAR(255) | NOT NULL |
| video_url | TEXT | Cloudinary URL |
| description | TEXT | Brief lesson summary |
| order_index | INTEGER | Default 0 |
