# DB Migration v1 - Initial Schema

## Tables

| Table | Description |
|-------|-------------|
| `database_migrations` | Migration version tracking |
| `users` | User accounts (student/instructor) |
| `courses` | Course info (title, category, difficulty, price) |
| `lessons` | Video lessons belonging to courses |
| `enrollments` | Student-course enrollment with progress |
| `payments` | SePay payment records |
| `likes` | User-course likes (many-to-many) |

## Notes
- All IDs use UUID with `gen_random_uuid()`
- `TIMESTAMPTZ` for all timestamps
- `ON DELETE CASCADE` on all foreign keys
- `pgcrypto` extension enabled for `gen_random_uuid()`
